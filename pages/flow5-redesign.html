Background changes
After I handed in the assignmnet, I researched a lot about split screens and the pros and cons about them. I found out that the most frequent reason designers use split screen is when their content changes topic. Therefore I changed the background to one color. The original idea was to split the screen so the user would be able to survey the page and find the navigation and the contact info in the footer more easily. I still kept the neutral, grey background because all the images add a lot of color already. Additionally, grey stands for intelligence, a word that i would like to be compared too.

Hover headlines
The users from my user test said that they could not distinguish the headline from the description when hovering each projct. Therefore, I increased the font size.

Alignment
Alignment is the A in the model C.R.A.P. I modified some of the flow pages by aligning the images and the text. I also changed the text margin on the right columm so it aligned with the project description.
